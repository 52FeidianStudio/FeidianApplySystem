# Feidian-Apply-System

## 基础

基于：React + Vite + Typescript + antd

### **拉取：**

```git
# GitHub
git clone https://github.com/52FeidianStudio/FeidianApplySystem
```

### **运行：**

```shell
npm install
//运行
npm run dev
//构建
npm run build
```

## 

##  🚀 功能

- 用户登录注册

- 通过邮箱找回密码

- 申请提交（表单校验，证件照上传）

- 查看面试结果

- 管理员可对申请表进行审核（通过邮件进行通知）

- 后台查看详细的用户数据

- 适配手机端布局
